Tentang Sistem

Teknologi yang Digunakan

Stack teknologi modern untuk membangun sistem deteksi berita palsu yang akurat dan efisien.

IndoBERT
Pre-trained BERT model untuk Bahasa Indonesia dari IndoNLU benchmark, digunakan sebagai backbone classifier.
PyTorch & HuggingFace
Framework deep learning dan Transformers library untuk fine-tuning dan inferensi model BERT.
Flask Python
Micro web framework Python untuk membangun REST API dan antarmuka web aplikasi deteksi berita.
Plotly Visualisasi
Library visualisasi interaktif untuk menampilkan distribusi data, confusion matrix, dan metrik performa.
Scikit-learn
Digunakan untuk evaluasi model: classification report, confusion matrix, dan metrik pengukuran performa.
Dataset Berita Indonesia
{{ dataset_info.total or 248 }} data dari TurnBackHoax.id dan Antara News dengan label FAKE/REAL yang terverifikasi.
